# Requirement
```
- python3
- adb
- scrcpy 
```
# How To Install
```bash
$ sudo apt-get install adb
$ git clone https://github.com/security007/sword.git
$ cd sword
$ chmod 755 install.x
$ ./install.x
$ sword
```

# Uninstall
```bash
$ cd sword
$ chmod 755 uninstall.x
$ ./uninstall.x
```
